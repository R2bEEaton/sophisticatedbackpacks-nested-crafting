
# Sophisticated Backpacks Advanced Crafting

NeoForge 1.21.1 addon for Sophisticated Backpacks and Sophisticated Core.

Adds `sophisticatedbackpacks:advanced_crafting_upgrade`, a crafting upgrade that refills crafting grid ingredients through the inception-aware upgrade-processing inventory. If an outer backpack has an inception upgrade, recipe refills can consume matching items from nested backpacks.
